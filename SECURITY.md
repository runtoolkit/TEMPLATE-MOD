# Security Policy

## Supported Versions

| Minecraft Version | Supported          |
|-------------------|--------------------|
| 26.1.2            | :white_check_mark: |
| < 26.1.1          | :x:                |

## Reporting a Vulnerability

**Do not open a public GitHub issue for security vulnerabilities.**

Report vulnerabilities privately via [GitHub's private vulnerability reporting](https://github.com/runtoolkit/TEMPLATE-MOD/security/advisories/new).

You can expect:
- An acknowledgement within **72 hours**
- A status update within **7 days**
- A patch release if the vulnerability is confirmed

Declined reports will receive a brief explanation.
